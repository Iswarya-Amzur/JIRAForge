# =============================================================================
# Platform-Specific Notes
# =============================================================================
# 
# Windows:
#   - All packages work out of the box
#   - Install Tesseract binary separately for tesseract engine
#   - GPU support: pip install paddlepaddle-gpu (requires CUDA)
#
# Linux (Ubuntu/Debian):
#   - Install tesseract-ocr: sudo apt-get install tesseract-ocr
#   - GPU support: pip install paddlepaddle-gpu (requires CUDA + cuDNN)
#   - Headless servers: sudo apt-get install xvfb
#
# macOS (Intel):
#   - Install tesseract: brew install tesseract
#   - CPU-only for PaddleOCR (no GPU support)
#   - All packages compatible with x86_64 architecture
#
# macOS (Apple Silicon M1/M2/M3):
#   - Install tesseract: brew install tesseract
#   - Optimized packages for ARM64 architecture
#   - PyTorch with MPS (Metal) acceleration support
#   - Use: arch -arm64 pip install <package> if needed
